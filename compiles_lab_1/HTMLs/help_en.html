Text Editor Help System

File

  • New — creates a new document in a tab.
  • Open — opens a text file from disk.
  • Save — saves the current document.
  • Save As… — saves the document under a new name.
  • Exit — closes the application.

Edit

  • Undo — reverts the last action.
  • Redo — repeats the reverted action.
  • Cut — cuts the selected text.
  • Copy — copies the selected text.
  • Paste — inserts text from the clipboard.
  • Delete — deletes the selected text.
  • Select All — selects the entire document.

Tabs

  • Close Tab — closes the current document.

Run

  • Start — runs text analysis and shows the result in the bottom panel.

Localization

  • Русский / English — switches the interface language.

Help

  • Help — opens this document.
  • About — information about the version and developer.
