Справочная система Text Editor

Файл

  • Создать — создаёт новый документ во вкладке.
  • Открыть — открывает текстовый файл с диска.
  • Сохранить — сохраняет текущий документ.
  • Сохранить как… — сохраняет документ под новым именем.
  • Выход — завершает работу программы.

Правка

  • Отменить — отменяет последнее действие.
  • Повторить — повторяет отменённое действие.
  • Вырезать — вырезает выделенный текст.
  • Копировать — копирует выделенный текст.
  • Вставить — вставляет текст из буфера обмена.
  • Удалить — удаляет выделенный текст.
  • Выделить всё — выделяет весь текст документа.

Вкладки

  • Закрыть вкладку — закрывает текущий документ.

Запуск

  • Пуск — запускает анализ текста и выводит результат в нижнюю панель.

Локализация

  • Русский / English — переключение языка интерфейса.

Справка

  • Вызов справки — открывает этот документ.
  • О программе — информация о версии и разработчике.
