Вариант задания №40
Объявление целочисленной константы с инициализацией (Kotlin)

Данная работа посвящена разработке автоматной грамматики и синтаксического анализатора для конструкции объявления целочисленной константы в стиле языка Kotlin. Конструкция включает ключевые слова const и val, идентификатор, указание типа Int, оператор присваивания и целочисленный литерал.

Примеры корректных входных данных:

const val x: Int = 10;
const val _number: Int = -5;
const val index123: Int = 0;
